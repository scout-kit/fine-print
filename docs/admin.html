Admin Dashboard

Manage projects, review photos, monitor the print queue, and configure printers — all from any device.

Dashboard Overview

At a glance, see how many photos are pending review, in progress, printed, or have issues. Click any stat to jump to the relevant view.

  • Pending Review — uploaded photos awaiting approval
  • In Progress — approved, queued, and currently printing
  • Printed — successfully printed photos
  • Issues — failed or rejected photos

Project Management

Create and manage multiple projects for different events or stations. Each project has its own template, visibility, and type.

  • Standard or Photo Booth project types
  • Public, hidden (link-only), or private visibility
  • QR codes and share links per project
  • Copy projects with all settings and templates
  • Per-project photo, template, and settings tabs

Printer Configuration

Detect and manage CUPS printers. No printers are enabled by default — the admin must explicitly enable each one.

  • Detect printers from CUPS
  • Enable/disable individual printers
  • Round-robin or manual assignment mode
  • 4x6 and postcard media sizes
  • Test print functionality

Photo Review

Click any photo to open the full modal — see the original, the print preview with template applied, and take action.

  • Toggle between original and print preview
  • Approve, reject, edit, download, delete
  • Reprint with or without template re-render
  • Set copies per photo at approval time
  • Filter by status or project
